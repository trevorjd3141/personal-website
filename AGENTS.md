# Agent notes — Personal Website

This is a **static single-page HTML site** — no framework, no build step, no
package manager. All content lives in `index.html`; custom styles in `style.css`.

Dependencies are loaded via CDN (Bootstrap 4.5, jQuery 3.5.1, Font Awesome,
Google Fonts). There is nothing to install.

See `CLAUDE.md` for project rules and content update conventions.
